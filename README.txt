Version ionic-cli :
  5.4.16
Version node : v9.11.2
Version npm : 6.4.1

Installation du projet,
commande à executer :
npm install leaflet --save
npm install @types/leaflet
npm install

Si un problème survient, il est très probable que ce soit dû aux sass et leaflet, lancer la commande suivante puis reprenez avec la commande qui était en erreur.
npm rebuild node-sass

Lancer le projet :
ionic serve
